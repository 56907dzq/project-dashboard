## 接口含义
在仪表盘综合管理页面中删除组件


## 请求参数

| 属性  | 说明         | 类型   | 默认值 |
| ----- | ----------- | ------ | ------ |
| widgetsname | 组件名称，对应新增中的name字段 | number|string | -    |


## 返回数据参数

返回数据整体格式参考[接口规范](#/guide/specification)中的请求成功接口。


res.data
空
